# Latent Representations of Predator-Prey Dynamics with a VAE

A Variational Autoencoder trained on windowed trajectory data from the Lotka-Volterra equations. The goal is for the VAE to discover, without supervision, a low-dimensional latent space whose axes correlate with the physical parameters of the ODE — in this case the prey growth rate α.

Trajectories are generated by sweeping α across several values while keeping all other parameters fixed. Short time-aligned windows are randomly sampled from each trajectory, normalized, and used to train the VAE. After training, the latent space is visualized to see whether the network has learned to separate trajectories by their underlying α value.

## Files

| File | Description |
|------|-------------|
| `schema.txt` | Full project schema: equations, parameters, architecture, and training config |
| `solve_lotka_volterra.py` | Solves the Lotka-Volterra ODE for several α values, samples random windows, normalizes, and saves the dataset to `predator_prey_data.pt` |
| `vae.py` | VAE model definition (encoder, decoder, reparameterization, loss, training loop) |
| `train_vae_lotkavolt.py` | Loads the dataset, trains the VAE, plots the loss curve, and saves weights to `vae_lotkavolt.pt` |
| `plot_latent_space.py` | Loads trained weights, encodes the dataset, and plots 2D and 3D latent space visualizations colored by α |

## Usage

```bash
# 1. Generate dataset
python solve_lotka_volterra.py

# 2. Train VAE
python train_vae_lotkavolt.py

# 3. Visualize latent space
python plot_latent_space.py
```
