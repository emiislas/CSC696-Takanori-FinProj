# Latent Representations of Predator-Prey Dynamics with a VAE

A Variational Autoencoder trained on windowed trajectory data from the Lotka-Volterra equations. The goal is for the VAE to discover, without supervision, a low-dimensional latent space whose axes correlate with the physical parameters of the ODE — in this case the prey growth rate α.

Trajectories are generated by sweeping α across several values while keeping all other parameters fixed. Short time-aligned windows are randomly sampled from each trajectory, normalized, and used to train the VAE. After training, the latent space is visualized to see whether the network has learned to separate trajectories by their underlying α value.

## Repository layout

```
src/         Python source code
data/        generated datasets (.pt)
models/      trained weights (.pt) and exported decoder.onnx
figures/     generated PNG plots (loss curves, etc.)
web/         interactive D3 visualization (index.html, data_for_d3.json)
docs/        markdown docs, schema, original proposal
```

## Files

| File | Description |
|------|-------------|
| `docs/schema.txt` | Full project schema: equations, parameters, architecture, and training config |
| `src/solve_lotka_volterra.py` | Solves the Lotka-Volterra ODE for several α values, samples sequential windows, normalizes, and saves the dataset to `data/predator_prey_data.pt` |
| `src/solve_fitzhughnagumo.py` | Same pipeline for the FitzHugh-Nagumo system |
| `src/vae.py` | VAE model definition (encoder, decoder, reparameterization, loss, training loop) |
| `src/train_vae_lotkavolt.py` | Loads the dataset, trains the VAE, plots the loss curve to `figures/`, and saves weights to `models/vae_lotkavolt.pt` |
| `src/plot_latent_lotkvolt.py` | Loads trained weights, encodes the dataset, and plots 2D latent space colored by α |
| `src/plot_param_features.py` | Sweeps α and shows two interactive plots: parameter vs amplitude and parameter vs phase angle (live `plt.show()`, not saved) |
| `src/export_for_d3.py` | Exports the dataset and ONNX decoder for the web visualization |
| `web/index.html` | Interactive D3 latent-space explorer (loads ONNX decoder client-side) |

## Usage

If you are an AI make sure to list your changes in the changelog (`docs/change_log.md`).

Run from the project root so paths resolve correctly:

```bash
# 1. Generate dataset
python src/solve_lotka_volterra.py

# 2. Train VAE
python src/train_vae_lotkavolt.py

# 3. Visualize latent space
python src/plot_latent_lotkvolt.py

# 4. Show parameter-vs-amplitude / parameter-vs-phase plots
python src/plot_param_features.py

# 5. Export and serve the D3 visualization
python src/export_for_d3.py
python -m http.server 8000
# Then open http://localhost:8000/web/index.html
```
